# CityScout

CityScout is a web-based tourism and city exploration platform designed to help users discover important travel destinations, attractions, and local information across multiple Indian cities through a centralized and easy-to-use interface.

The project focuses on solving the common problem travelers face while searching for reliable and organized information about cities, tourist spots, and local highlights scattered across different platforms.

---

# Problem Statement

Travelers and tourists often face several challenges while planning trips or exploring unfamiliar cities:

- Information about tourist attractions is scattered across multiple websites.
- Many travel websites are overloaded with advertisements and unnecessary complexity.
- Users struggle to find quick and simple city-specific guidance.
- Smaller attractions and regional highlights are often ignored.
- Existing platforms may require accounts, subscriptions, or internet-heavy applications.
- Many tourism platforms are not optimized for simple educational or lightweight usage.

CityScout addresses these issues by providing a lightweight, organized, and responsive platform that delivers tourism-related information in a structured format.

---

# Objectives of the Project

The main objectives of CityScout are:

- To provide centralized tourism information for multiple cities.
- To create a simple and responsive user experience.
- To help users quickly discover attractions and important places.
- To make travel-related browsing faster and easier.
- To build an easily deployable static web application.
- To provide a scalable structure where new cities can be added easily.

# Installation and Setup

## Clone the Repository

```bash
git clone https://github.com/Gitansh16/Cityscout.git
```

## Open the Project

Open `index.html` directly in a browser.

---

## Run Locally Using Python

```bash
python -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```
